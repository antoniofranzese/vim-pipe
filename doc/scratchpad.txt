INTRODUCTION                                    *scratchpad*

Scratchpad sets up a REPL.

CONFIGURATION                                   *scratchpad-configuration*

                                                *scratchpad-b:scratchpad_command*
b:scratchpad_command    Set this to a shell command that will take the
			            contents of your buffer on STDIN, and return the
                        result on STDOUT.

ABOUT                                           *scratchpad-about*
